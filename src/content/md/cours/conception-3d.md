La modélisation et l'impression 3D permettent de passer d'une idée à un objet physique. Cette technologie est utilisée dans de nombreux domaines : industrie, médecine, architecture, mode...

---

## De l'idée à l'objet

### Le cahier des charges

Avant de modéliser, il faut définir précisément ce que l'on veut créer :

- **Fonctions** : À quoi sert l'objet ? Quels problèmes résout-il ?
- **Contraintes** : Dimensions, résistance, esthétique
- **Matériaux** : PLA, PETG, ABS... selon l'usage

### Du croquis au modèle 3D

1. **Croquis à main levée** : Explorer les idées rapidement
2. **Dessin technique** : Définir les dimensions précises
3. **Modélisation 3D** : Créer le fichier numérique

---

## Les formes de base (primitives)

En modélisation 3D, on part de formes simples appelées **primitives** :

| Primitive | Usages courants |
|-----------|-----------------|
| Cube / Pavé | Boîtes, supports, structures |
| Cylindre | Axes, tubes, poignées |
| Sphère | Boutons, articulations |
| Cône | Embouts, guides |

### Les opérations booléennes

Pour créer des formes complexes, on combine les primitives :

- **Union** : Fusionner deux formes
- **Soustraction** : Creuser une forme dans une autre
- **Intersection** : Garder uniquement la partie commune

## Modélisation avec Tinkercad

### Interface de Tinkercad

Tinkercad est un outil en ligne gratuit et simple pour débuter :

1. Glisser-déposer les formes sur le plan de travail
2. Redimensionner avec les poignées
3. Utiliser les formes "trous" pour creuser

### Bonnes pratiques

- Travailler sur un plan de travail stable
- Grouper les éléments régulièrement
- Nommer ses projets clairement
- Exporter en STL pour l'impression

## L'impression 3D

### Principe de fonctionnement

L'impression 3D FDM (dépôt de fil fondu) :

1. Le filament plastique est chauffé
2. Il est extrudé par une buse
3. La pièce est construite couche par couche

### Paramètres essentiels

| Paramètre | Description | Impact |
|-----------|-------------|--------|
| Hauteur de couche | Épaisseur de chaque couche (0.1-0.3mm) | Qualité vs vitesse |
| Remplissage | Pourcentage de matière à l'intérieur (10-100%) | Solidité vs poids |
| Supports | Structures temporaires pour les surplombs | Complexité des formes |
| Température | Selon le filament (PLA: 200°C, PETG: 240°C) | Adhésion et qualité |

### Les filaments courants

- **PLA** : Facile, biodégradable, peu résistant à la chaleur
- **PETG** : Plus solide, résistant à l'eau
- **ABS** : Résistant aux chocs, nécessite ventilation

## Contrôle qualité

### Vérifications avant impression

- Dimensions conformes au cahier des charges
- Pas de surfaces flottantes (non supportées)
- Fichier STL sans erreurs

### Vérifications après impression

- Mesures avec un pied à coulisse
- Test de solidité selon l'usage prévu
- Ajustements si nécessaire (ponçage, assemblage)

## Activité pratique

### Projet : Créer un objet utile

1. **Identifier un besoin** : Un problème du quotidien à résoudre
2. **Rédiger le cahier des charges** : Fonctions, contraintes, dimensions
3. **Réaliser des croquis** : Explorer plusieurs solutions
4. **Modéliser avec Tinkercad** : Créer le fichier 3D
5. **Imprimer et tester** : Valider la conception

### Critères d'évaluation

- L'objet répond-il au besoin identifié ?
- Les dimensions sont-elles respectées ?
- La qualité d'impression est-elle satisfaisante ?
- L'objet est-il fonctionnel ?
